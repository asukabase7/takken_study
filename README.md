# 📚 宅建試験 学習ツール (Takken Study App)

PythonによるスクレイピングからGUIアプリケーション構築までを一貫して行った、宅建試験対策用のデスクトップアプリケーションです。
過去問の収集から、学習履歴のトラッキング、生成AIへのシームレスな質問機能まで、学習者の効率を最大化するための機能を備えています。

## ✨ 特徴 (Features)

*   **🌐 過去問の自動収集とデータクレンジング**
    *   Webスクレイピング技術により、直近10年分以上の過去問データを自動的に収集。
    *   収集データに含まれる「和暦（令和・平成）」などの表記揺れを正規化し、純粋な「西暦」へ統一するデータクレンジング処理を実装。
*   **💻 快適なデスクトップUI**
    *   Tkinterを活用し、長時間の学習でも目が疲れにくい「ダークテーマ」を採用。
    *   ウィンドウの拡大縮小に追従する、レスポンシブで視認性の高いレイアウト。
*   **📂 動的な問題フィルタリング**
    *   「年度別」および「分野別（宅建業法、法令上の制限、権利関係など）」で、解きたい問題をピンポイントに絞り込み可能。
*   **📊 学習履歴と統計トラッキング**
    *   ユーザーの回答（正誤）をデータベースに記録し、リアルタイムに「全体の正答率」をUI上で計算・表示。
    *   **復習モード**: 直近の回答で間違えた問題のみを抽出し、弱点克服をサポート。
    *   **⭕❌ 一問一答モード**: 通常の4択問題をバラし、1肢ごとの正誤判定を行うスピード学習モード。
*   **🤖 AI学習アシスト連携**
    *   間違えた問題や解説が理解できない問題に対して、LLM（GeminiやChatGPTなど）に「なぜこの選択肢が間違いなのか」を質問するための専用プロンプトをワンクリックで生成＆クリップボードへコピー。

## 🛠️ 技術スタック (Tech Stack)

*   **言語**: Python 3.12
*   **データベース**: SQLite3 (リレーショナルデータの保存・抽出、ウィンドウ関数を用いた履歴分析)
*   **GUIライブラリ**: Tkinter / ttk (Custom Styles)
*   **スクレイピング・データ収集**: `requests`, `BeautifulSoup4`

## ⚠️ 注意事項 (Note)

本アプリケーションで使用している過去問のテキストデータ等はそれぞれの著作権元に帰属します。
また、ファイルサイズおよびライセンスへの配慮から、収集済みのデータベースファイル (`.db`) は本リポジトリの管理対象（`.gitignore`）から除外しています。
